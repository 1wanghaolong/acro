# Admin
## 安装依赖
`
yarn install
`
## 运行
#### CMS
`
yarn run dev:cms
`
#### TRS
`
yarn run dev:trs
`
#### WEALTH
`
yarn run dev:wealth
`
## 打包
### 测试
#### CMS
`
yarn run build:cms_dev
`
#### TRS
`
yarn run build:trs_dev
`
#### WEALTH
`
yarn run build:wealth_dev
`
### 生产
#### CMS
`
yarn run build:cms_prod
`
#### TRS
`
yarn run build:trs_prod
`
#### WEALTH
`
yarn run build:wealth_prod
`
